## ❓ СТРУКТУРА УСПЕШНЫХ КВИЗОВ (выявлено)

### Обязательные вопросы (РАБОТАЮТ):

**Q1: Уровень опыта**
- Никогда не пел / Пел в школе / Опытный певец
- **Почему:** сегментирует по готовности, снижает возражения

**Q2: Мотивация / Боль**
- Стеснение / "Нет слуха" / Зажимы / Время / Выступление
- **Почему:** выявляет психологическую боль, резонирует с клиентом

**Q3: Стиль музыки**
- Поп / Джаз / Классика / Рок / Другое
- **Почему:** сегментирует по личности, укрепляет идентичность

**Q4: Удобство / Практика**
- Время / Расписание / Формат
- **Почему:** преодолевает практические возражения

**Q5: Контакт** (обязательный)
- Имя + Телефон
