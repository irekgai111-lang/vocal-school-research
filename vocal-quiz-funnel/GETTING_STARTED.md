## 1️⃣ ПОЛУЧИТЬ BOT_TOKEN

### Шаг 1: Откройте Telegram

### Шаг 2: Найдите @BotFather

Поиск в Telegram → "@BotFather" → Открыть

### Шаг 3: Отправьте `/newbot`

В чате с @BotFather отправьте сообщение:
```
/newbot
```

### Шаг 4: Назовите бота

@BotFather спросит: "Как назвать бота?"

Ответьте:
```
Твой Голос Bot
```

(или любое другое имя на русском/английском)

### Шаг 5: Выберите username

@BotFather спросит: "Выберите username для бота"

Ответьте (латиница, уникальное имя):
```
tvoygolosstudia_bot
```

или

```
tvoygolossbot
```

### Шаг 6: Скопируйте токен

@BotFather вернет:
```
Done! Congratulations on your new bot. You will find it at t.me/tvoygolossbot.
You can now add a description, about section and profile picture for your bot, see /help for a list of commands.

Here's your token: 123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11

For a description of the Bot API, open this page: https://core.telegram.org/bots/api

Keep your token secure and store it safely!
```

**Скопируйте токен:** `123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`

## ❓ ВОЗНИКЛА ПРОБЛЕМА?

### Бот не запускается

```powershell
# Проверьте Python
python --version

# Переустановите зависимость
pip install --upgrade python-telegram-bot

# Посмотрите ошибку в окне PowerShell
```

### Бот запускается, но не отвечает

- Убедитесь, что вы отправляете сообщения именно вашему боту
- Может потребоваться 10-30 секунд на инициализацию

### Команды не работают

- Убедитесь, что сообщение начинается с `/` (слэш)
- Попробуйте /start еще раз
