### ДЕ НЬ 2️⃣ (2 часа): НАСТРОЙКА TELEGRAM-БОТА

```
ШАГ 1: Получить токен бота
├─ Отправить /newbot в @BotFather
├─ Назвать: "Твой Голос Bot"
├─ Получить TOKEN (выглядит: "123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11")
└─ Скопировать в переменную BOT_TOKEN

ШАГ 2: Получить свой Telegram ID
├─ Отправить /start в @userinfobot
├─ Скопировать ваш ID (число, например: 987654321)
└─ Вставить в переменную ADMIN_CHAT_ID

ШАГ 3: Установить зависимости
$ pip install python-telegram-bot

ШАГ 4: Скопировать файл 10_BOT-UPDATED-WITH-BOOKING.py
├─ Заменить BOT_TOKEN на ваш
├─ Заменить ADMIN_CHAT_ID на ваш
└─ Сохранить

ШАГ 5: Протестировать бот
$ python 10_BOT-UPDATED-WITH-BOOKING.py
└─ Должно вывести: "✅ Бот запущен. Нажмите Ctrl+C для остановки."

ШАГ 6: Протестировать команды
├─ В Telegram отправить /start боту
├─ Отправить /help
├─ Отправить /book
└─ Проверить, что все работает

✅ РЕЗУЛЬТАТ: Бот работает! Можно оставить в фоне или на сервере
```

## 📞 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Бот не запускается:
```bash
# Проверить Python версию
python --version  # Должна быть 3.8+

# Переустановить зависимости
pip install --upgrade python-telegram-bot

# Проверить логи
cat bot_logs.log
```

### Квиз не сохраняет данные:
```
1. Проверить интеграцию в Marquiz (Settings → Integrations)
2. Убедиться, что вебхук правильно настроен
3. Вручную отправить JSON тестовое сообщение боту
```

### Клиенты не видят форму выбора времени:
```
1. Проверить, что форма добавлена в результат квиза
2. Протестировать с другого браузера (может быть кэш)
3. Проверить, что форма правильно заполнена в Marquiz
```
