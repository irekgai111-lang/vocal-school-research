# 🚀 ФИНАЛЬНЫЙ ПЛАН: 7 ДНЕЙ ДО ПЕРВЫХ КОНТАКТОВ

**Версия:** ОФЛАЙН-ОПТИМИЗИРОВАННАЯ  
**Воронка:** Квиз → Контакт → Пробное ОФЛАЙН → Оплата + бонус  
**Результат:** 50-100 контактов, 30-50 пробных, 20-35 оплат  

---

## 📅 ДЕНЬ 1 (СЕГОДНЯ): КВИЗ + ПОДГОТОВКА

### ✅ УТРО: Создание Typeform квиза (45 мин)

1. Зайди на **typeform.com** → Регистрация (5 мин)
2. Нажми "Create a new form"
3. **Добавь 5 вопросов** (из файла `TYPEFORM_SETUP_GUIDE.md`):
   - Q0: Фильтр платежеспособности (ГЛАВНЫЙ!)
   - Q1: Главный страх
   - Q2: Скрытое желание
   - Q3: Уровень опыта
   - Q4: Практический барьер
4. **Добавь контактную форму** (имя + телефон + email + время)
5. **Оформи красиво** (фиолетовые/розовые цвета)
6. **Финальные экраны**:
   - Для "Да" → Спасибо, запись на урок
   - Для "Нет" → Спасибо, до встречи
7. **Получи ссылку** на квиз (вид: typeform.com/to/xxxxxxx)
8. **Протестируй** (пройди все варианты)

✅ **Результат:** Квиз готов, ссылка скопирована

### ✅ ДЕНЬ: Интеграция с Zapier (15 мин)

1. Зайди на **zapier.com** → Регистрация
2. Create a Zap → Trigger: Typeform + "New Entry"
3. Action: Telegram → "Send Message"
4. Оформи сообщение:
   ```
   📝 НОВЫЙ КОНТАКТ:
   Имя: {Name}
   Телефон: {Phone}
   Email: {Email}
   ```
5. Test & Create

✅ **Результат:** Контакты из квиза идут в Telegram автоматически

### ✅ ВЕЧЕР: Подготовка файлов (30 мин)

1. Скачай файлы:
   - `TYPEFORM_SETUP_GUIDE.md` (для справки)
   - `OFFLINE_FUNNEL_STRATEGY.md` (для понимания)
   - `03_telegram_bot_OFFLINE.py` (переработанный бот)

2. Сохрани в папку: `C:\Users\Dell\vocal-quiz-funnel\`

✅ **ИТОГ ДНЯ 1:** Квиз работает, Zapier настроен

## 📅 ДЕНЬ 2: БОТА НА ЛОКАЛЬНОЙ МАШИНЕ

### ✅ УТРО: Установка Python (30 мин)

1. Скачай **Python 3.8+** с **python.org**
2. Установи (отметь "Add Python to PATH")
3. Проверь установку:
   ```bash
   python --version
   ```
   (должно показать версию, например: Python 3.11.0)

✅ **Результат:** Python установлен

### ✅ ДЕНЬ: Получение токенов Telegram (20 мин)

**Шаг 1: BOT_TOKEN от BotFather**
1. Открой Telegram
2. Напиши @BotFather
3. Команда: `/newbot`
4. Следуй инструкциям:
   - Имя: "Vocal School Quiz Bot" (или любое)
   - Username: `vocal_school_bot` (должен быть уникальный)
5. BotFather отправит **BOT_TOKEN** вида:
   ```
   123456789:ABCdefGHIjklmnoPQRstuvWXyzABCdef
   ```
6. Скопируй токен (сохрани в безопасное место!)

✅ **Результат:** BOT_TOKEN получен

**Шаг 2: ADMIN_ID**
1. Открой Telegram
2. Напиши @userinfobot
3. Боб отправит:
   ```
   User id: 1234567890
   ```
4. Это твой **ADMIN_ID** (скопируй)

✅ **Результат:** ADMIN_ID получен

### ✅ ВЕЧЕР: Установка бота (30 мин)

1. Открой Command Prompt или PowerShell
2. Перейди в папку проекта:
   ```bash
   cd C:\Users\Dell\vocal-quiz-funnel
   ```
3. Установи библиотеку:
   ```bash
   pip install python-telegram-bot
   ```
4. Открой файл `03_telegram_bot_OFFLINE.py`
5. Найди строки (вверху):
   ```python
   BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"
   ADMIN_ID = 123456789
   PAYMENT_LINK = "https://your-payment-link.com"
   SUPPORT_USERNAME = "YourUsername"
   SUPPORT_PHONE = "+7 (999) 123-45-67"
   ```
6. Вставь реальные значения:
   ```python
   BOT_TOKEN = "123456789:ABCdefGHIjklmnoPQRstuvWXyzABCdef"
   ADMIN_ID = 1234567890
   PAYMENT_LINK = "https://payment.link.com"  # Или оставь заглушку
   SUPPORT_USERNAME = "YourUsername"
   SUPPORT_PHONE = "+7 (999) 123-45-67"
   ```
7. Сохрани файл (Ctrl+S)

✅ **Результат:** Бот готов к запуску

## 📅 ДЕНЬ 3: ЗАПУСК И ТЕСТИРОВАНИЕ

### ✅ УТРО: Запуск бота (5 мин)

1. Открой Command Prompt в папке проекта
2. Запусти:
   ```bash
   python 03_telegram_bot_OFFLINE.py
   ```
3. Должно вывести:
   ```
   🚀 Бот запущен!
   📬 Система рассылки активирована (дни 0, 1, 3, 5, 7)
   ```

✅ **Результат:** Бот работает локально

### ✅ ДЕНЬ: Тестирование бота (30 мин)

1. Открой Telegram
2. Напиши своему боту (@vocal_school_bot):
   ```
   /start
   ```
3. Пройди весь диалог:
   - Введи имя
   - Отправь номер телефона
4. Проверь:
   - ✅ Бот ответил приветствием?
   - ✅ Попросил имя?
   - ✅ Попросил телефон?
   - ✅ Контакт сохранился в `contacts.json`?
   - ✅ Админ получил уведомление?

✅ **Результат:** Бот полностью работает

### ✅ ВЕЧЕР: Интеграция квиз → бот (15 мин)

В Typeform, в **финальном экране** добавь ссылку на бота:

```
🎤 Запишись на урок:
👉 t.me/vocal_school_bot 👈

Бот сразу отправит тебе всё нужное!
```

✅ **ИТОГ ДНЯ 3:** Квиз + Бот + Запер интегрированы

## 📅 ДЕНЬ 4: ПОИСК ГРУПП

### ✅ УТРО-ДЕНЬ: Поиск групп (4 часа)

1. Открой Telegram
2. Используй поиск (иконка с лупой)
3. Ищи по словам:
   - "вокал"
   - "пение"
   - "саморазвитие"
   - "для женщин"
   - "красота"
   - "здоровье"

4. Критерии хорошей группы:
   - ✅ 500+ участников
   - ✅ Посты каждый день
   - ✅ Можно писать (не закрытая)
   - ❌ Не спам

5. Создай таблицу:
   ```
   # Группы для постинга
   
   | Название | Ссылка | Участники | Активность |
   |----------|--------|-----------|-----------|
   | Вокал для всех | @vocal_for_all | 1200 | высокая |
   | ... | | | |
   ```

**НАЙДИ 15-20 ГРУПП**

✅ **Результат:** Список групп готов

### ✅ ВЕЧЕР: Подготовка постов (1 час)

Из файла `04_telegram_strategy.md` скопируй 5 вариантов постов:

1. **Пост #1: Нейтральный** (для админа)
2. **Пост #2: Контентный** (для органика)
3. **Пост #3: Провокационный** (для внимания)
4. **Пост #4: Социальный** (эмпатия)
5. **Пост #5: Срочность** (FOMO)

В каждом посте замени:
- `[ССЫЛКА]` → на ссылку твоего квиза
- `@bot_username` → на имя своего бота

Сохрани в Google Docs или Word

✅ **ИТОГ ДНЯ 4:** 15 групп + 5 вариантов постов

## 📅 ДЕНЬ 5-6: БОЛЬШОЙ ПОСТИНГ

### ✅ ДЕНЬ 5: Первая волна (5-6 часов)

**Расписание:**

```
🕙 10:00 → Группы 1-3 (Пост #1)
⏸️ Пауза 10 мин
🕙 10:30 → Группы 4-6 (Пост #2)
⏸️ Пауза 10 мин
...
```

**Процесс для каждой группы:**
1. Открой группу в Telegram
2. Нажми "Написать сообщение" (на названии группы)
3. Скопируй пост из Google Docs
4. Вставь в поле сообщения
5. Отправь в комментарии (если требует админа - пиши админу)
6. Переходи к следующей группе (интервал 10 мин)

**Система чередования постов:**
- Группы 1-3: Пост #1
- Группы 4-6: Пост #2
- Группы 7-9: Пост #3
- Группы 10-12: Пост #4
- Группы 13-15: Пост #5

✅ **Результат:** 15 постов в группах

### ✅ ДЕНЬ 6: Повторяем (5-6 часов)

Повторяем то же самое в тех же группах с ДРУГИМИ постами:
- Новый раунд: Пост #1, #2, #3, #4, #5

✅ **Результат:** 30 постов за 2 дня = 50-100 контактов ожидается
