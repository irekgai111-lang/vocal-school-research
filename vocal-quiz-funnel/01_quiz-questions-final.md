## 📊 ИНТЕГРАЦИЯ С РЕЗУЛЬТАТАМИ

Каждый ответ связан с одним из **5 сегментов**:

```
Q0 (фильтр) → Q1 (боль) + Q2 (желание) + Q3 (опыт) + Q4 (возражение) = СЕГМЕНТ + РЕЗУЛЬТАТ
```

**Алгоритм сегментации:**

**ЕСЛИ Q0 = C (только бесплатно):** Квиз заканчивается → контакт не собираем

**ЕСЛИ Q0 = A или B:**

1. **МЕЧТАТЕЛЬ СО СТРАХОМ** → Q1: A или C + Q2: D
   → Результат: "Ты ищешь СВОБОДУ"

2. **ТРАВМИРОВАННЫЙ НОВИЧОК** → Q1: B + Q3: A
   → Результат: "Пора переписать историю"

3. **ПРИВАТНЫЙ ПЕВЕЦ** → Q1: D + Q3: C
   → Результат: "Ты уже поёшь"

4. **ЗАЖАТЫЙ ОПЫТНЫЙ** → Q3: D или E + Q4: E
   → Результат: "Тебе нужна техническая коррекция"

5. **РАЗНООБРАЗНИК** → Q4: E + Q2: D
   → Результат: "Ты ищешь РАЗНООБРАЗИЯ"

```
Спасибо, [Имя]! 💜

Готов помочь тебе начать петь правильно!

Оставь координаты для пробного урока:
```

**Форма содержит:**

| Поле | Обязательное | Зачем |
|------|--------------|-------|
| Имя | ✅ Да | Персонализация |
| Телефон / Telegram | ✅ Да | Контакт |
| Email | ✅ Да | Альтернативный контакт |
| Удобное время | ❓ Опционально | Предварительное расписание |

**Кнопка:** "🎤 Записаться на урок" → Отправка в CRM/Telegram-бот

## 🎯 СОВЕТ ДЛЯ РЕАЛИЗАЦИИ

✅ **На Marquiz:**
- Настроить условные переходы (branching logic)
- Q0 → определяет 80% конверсии (фильтр мусора)
- Q1 → определяет сегментацию
- Q3 → уточняет рекомендацию
- Результат считает автоматически
- Форма отправляется в Telegram-бот

✅ **На Quiz Please:**
- Использовать тэги (tags) для каждого варианта
- Q0 = C → финальный экран "Спасибо"
- Q0 = A/B → форма записи

✅ **На LeadQuizzes:**
- Drag-and-drop редактор
- Видео в Q1-Q3 (повышает энгейджмент)
- Форма интегрируется с Zapier → Telegram
