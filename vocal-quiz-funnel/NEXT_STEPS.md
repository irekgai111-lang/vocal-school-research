## 📅 ДЕНЬ 3

### ✅ Утро: Подготовка бота (2 часа)

**Требуется:**
1. Python 3.8+ (скачай с python.org)
2. Telegram: @BotFather (получи BOT_TOKEN)
3. Telegram: @userinfobot (получи свой ID = ADMIN_ID)

**Шаги:**
1. Открой `03_telegram_bot.py`
2. Найди строки:
   ```python
   BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"  # Вставь здесь
   ADMIN_ID = 123456789  # Вставь здесь
   ```
3. Вставь реальные значения
4. Установи библиотеку:
   ```bash
   pip install python-telegram-bot
   ```
5. Запусти бот:
   ```bash
   python 03_telegram_bot.py
   ```

**ДЕЙСТВИЕ:**
- [ ] Python установлен
- [ ] BOT_TOKEN получен
- [ ] ADMIN_ID получен
- [ ] Значения вставлены в код
- [ ] python-telegram-bot установлена

## 🚨 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ?

### Квиз не открывается
- Проверь ссылку (скопируй правильно)
- Протестируй на другом браузере/телефоне
- Свяжись с поддержкой Marquiz

### Бот не отвечает
- Убедись, что Python работает (в консоли нет ошибок)
- Проверь токен (должен быть корректным)
- Перезагрузи бота

### Нет кликов с Telegram
- Убедись, что ссылка правильная (открывается)
- Попробуй другой текст поста (вариант 2, 3, 4)
- Посты в побольше групп (не только 5)

### Контакты не сохраняются
- Проверь, что форма правильно заполнена
- Посмотри `contacts.json` (должен быть создан)
- Если пусто — пересохраняй форму в боте
