```python
# Бот получает:
{
    "quiz_id": "quiz_001_tvoygolosstudia",
    "user_name": "[имя из формы]",
    "user_phone": "[телефон из формы]",
    "user_email": "[email из формы]",
    "quiz_result": "[номер результата 1-5]",
    "selected_time": "2026-05-15 18:30",
    "timestamp": "2026-05-09 14:23:45"
}

# БОТ ДЕЛАЕТ:
1️⃣ Сохраняет данные в contacts.json с пометкой "QUIZ_BOOKED"
2️⃣ Отправляет клиенту в Telegram:
   "Отлично! Вы записаны на [время]. Вот адрес..."
3️⃣ Отправляет ВАМ в Telegram:
   "[Имя] записался(ась) на [время]. Результат: [тип]"
4️⃣ Ставит напоминания (6, 4, 2 часа перед уроком)
```
