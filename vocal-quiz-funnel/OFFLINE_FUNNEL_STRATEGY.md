# 🎤 ПЕРЕРАБОТАННАЯ ВОРОНКА: ОФЛАЙН-МОДЕЛЬ

**Основано на:** Квиз → Контакт → Бесплатный пробный урок ОФЛАЙН → Оплата + 2 урока в подарок

---

## 📊 ТА ВОРОНКА (БЫЛ ОНЛАЙН-ОРИЕНТИРОВАН)

```
Квиз → Видео-диагностика → Платная покупка курса
```

❌ **Проблема:** Ты же приглашаешь на ОФЛАЙН! Видео — это лишний шаг.

---

## 📊 НОВАЯ ВОРОНКА (ОФЛАЙН-ОПТИМИЗИРОВАННАЯ) ✅

```
Квиз → БЕСПЛАТНОЕ пробное занятие ОФЛАЙН 
→ Оплата в день пробного (пакет 8 уроков + 2 в подарок) 
→ Итого 10 уроков
```

---

## 🤖 ПЕРЕРАБОТАННАЯ РАССЫЛКА В TELEGRAM-БОТЕ

**Вместо видео-диагностики → Напоминания о пробном занятии**

### **ДЕНЬ 0: СПАСИБО + ВЫЗОВ НА ПРОБНОЕ**

```
Текст:
🎉 Спасибо, [ИМЯ]!

Я проверила твои ответы 🎯

Вот что я вижу: ты именно та, кому нужна эта программа.

✅ Вот что дальше:

1️⃣ Запиши БЕСПЛАТНОЕ пробное занятие
   (Реальный голос, реальные результаты за 60 минут)

2️⃣ Приходи в студию и увидишь сам(а)

3️⃣ Если понравилось → Оплати в день пробного и получи:
   💝 Пакет из 8 уроков
   💝 + 2 урока в подарок (итого 10 уроков!)

🎤 Запишись на урок: /book
```

**Кнопка:** `/book` → бот предлагает выбрать время

---

### **ДЕНЬ 1: МОТИВАЦИЯ + НАПОМИНАНИЕ**

```
Текст:
🌟 [ИМЯ], твой пробный урок завтра/сегодня! 

Я уверена, что тебе понравится 🎵

Что ты получишь за 60 минут:
✅ Диагностика твоего голоса (прямо на уроке)
✅ Первые упражнения (ты сразу почувствуешь разницу)
✅ Индивидуальный план развития

Помни: первый урок БЕСПЛАТНЫЙ! 

Если не можешь прийти → напиши /reschedule
```

---

### **ДЕНЬ 3: РЕЗУЛЬТАТЫ + ТОЛЧОК К ОПЛАТЕ**

```
Текст:
💪 [ИМЯ]!

Как прошёл твой пробный урок? 🎤

Если прошёл:
✅ Отлично! При оплате в день пробного:
✅ Пакет из 8 уроков + 2 урока в подарок!

Ссылка для оплаты: /pay

Если не пришёл:
❓ Что случилось? Могу я чем-то помочь?
📞 Напиши /support или позвони: +7 (999) 123-45-67
```

### **ДЕНЬ 5: СРОЧНОСТЬ**

```
Текст:
⏰ [ИМЯ], ВНИМАНИЕ! 

Специальное предложение действует ещё 2 дня! 

При оплате в день пробного:
✅ Пакет из 8 уроков + 2 урока в подарок (итого 10!)

После 2 дней предложение закончится.

Оплати сейчас: /pay
```

---

### **ДЕНЬ 7: ФИНАЛЬНОЕ ПРЕДЛОЖЕНИЕ**

```
Текст:
🎯 [ИМЯ], это последний день!

Специальное предложение заканчивается СЕГОДНЯ в 23:59

Сегодня при оплате в день пробного:
✅ Пакет из 8 уроков + 2 урока в подарок (итого 10!)

После сегодня предложение закончится.

Начни свой путь в вокале СЕЙЧАС! 🚀

Оплати: /pay
Вопросы: /support
```

## 💰 СХЕМА ОПЛАТЫ

**За пробный урок:** Бесплатно ✅

**При оплате в день пробного:**
- Пакет из 8 уроков
- + 2 урока в подарок (итого 10 уроков)

## 📱 КНОПКИ В БОТЕ

### `/start` → Начало
- Привет! [ИМЯ], спасибо за прохождение квиза!
- Кнопки:
  - 📅 /book — Записаться на пробное
  - ❓ /help — Справка
  - 👩‍🏫 /support — Контакты

### `/book` → Запись на пробное
```
Выбери удобное время:
🕙 Утро (9:00–12:00)
🕐 День (12:00–17:00)
🕕 Вечер (17:00–21:00)
📆 Какой день?

→ Система сохраняет время
→ Отправляет админу уведомление
```

### `/pay` → Оплата
```
💳 Скидка 30%: 3500₽ + 2 урока в подарок

Варианты оплаты:
1️⃣ Перевод на карту
2️⃣ Яндекс.Касса
3️⃣ СБП (Система быстрых платежей)

[ССЫЛКИ НА ПЛАТЕЖИ]

Вопросы? /support
```

### `/reschedule` → Перенос пробного
```
Хочешь перенести пробное занятие?

Какая дата тебе удобна?
[КАЛЕНДАРЬ]

Что случилось?
- Работа
- Болезнь
- Другое

Мы перенесём! 💜
```

```
👩‍🏫 Директор: @YourUsername
📞 Телефон: +7 (999) 123-45-67
✉️ Email: info@vocalschool.ru

Ответим в течение 1 часа! 😊
```
