# 📋 TYPEFORM КВИЗ: ПОШАГОВАЯ ИНСТРУКЦИЯ

**Платформа:** Typeform (бесплатный план)  
**Время создания:** 30-45 мин  
**Результат:** Красивый, интерактивный квиз с логикой  

---

## 🚀 ШАГ 1: РЕГИСТРАЦИЯ (5 мин)

1. Зайди на **typeform.com**
2. Нажми "Sign up free"
3. Создай аккаунт (email + пароль)
4. Подтверди email

✅ **Готово!** Ты в админ-панели Typeform

---

## ✏️ ШАГ 2: СОЗДАНИЕ КВИЗА (35 мин)

### **Нажми "Create a new form"**

#### **ВОПРОС 0: ФИЛЬТР ПЛАТЕЖЕСПОСОБНОСТИ (ВАЖНЕЙШИЙ!)**

```
Тип: Multiple choice (одиночный выбор)

Заголовок:
💰 Вы готовы инвестировать в развитие своего вокала?

Варианты:
A) ✅ Да, готова платить — это для меня важно
B) ❓ Может быть, если будет результат — хочу попробовать
C) ❌ Нет, только бесплатно — ищу бесплатные уроки

⚙️ ЛОГИКА:
- Если выбрал C → Перейти на ФИНАЛЬНЫЙ ЭКРАН "Спасибо, до встречи"
- Если выбрал A или B → Продолжить квиз
```

**В Typeform:**
- Тип: "Multiple choice"
- Разные цвета для вариантов (красный для C, зелёный для A)
- Включить логику: Settings > Logic > Add rule

#### **ВОПРОС 1: ГЛАВНЫЙ СТРАХ**

```
Тип: Multiple choice

Заголовок:

Варианты:
A) Боюсь выглядеть смешной / стесняюсь своего голоса
```

## 🎨 ШАГ 3: ОФОРМЛЕНИЕ (10 мин)

### **Внешний вид:**

1. **Цвета:** Settings > Appearance
   - Background: Светло-фиолетовый (#F5E6FA) или светло-розовый (#FFF0F5)
   - Кнопки: Яркий фиолетовый (#9333EA) или розовый (#EC4899)

2. **Логотип:** Загрузи логотип вокальной школы (Settings > Logo)

3. **Шрифт:** Используй читаемый шрифт (по умолчанию хорош)

4. **Музыка (опционально):** Settings > Background music (загрузи музыку)

---

## 🔗 ШАГ 4: ИНТЕГРАЦИЯ (5 мин)

### **Zapier → Telegram Bot**

**Зачем:** Когда кто-то заполнит квиз → автоматически идёт контакт в бота

1. Зайди на **zapier.com** (бесплатный план)
2. Create a Zap
3. Trigger: Typeform → "New Entry"
4. Выбери свой квиз
5. Action: Telegram → "Send Message"
6. Выбери своего бота
7. Сообщение:
   ```
   📝 НОВЫЙ КОНТАКТ ИЗ КВИЗА:
   
   Имя: {Name}
   Телефон: {Phone}
   Email: {Email}
   Предпочитаемое время: {Time}
   ```
8. Нажми "Test" → "Create"

✅ **Готово!** Теперь контакты из квиза идут в Telegram автоматически
