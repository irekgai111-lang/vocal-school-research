## 🧠 ПРИНЦИПЫ ПСИХОЛОГИИ ПРОДАЖ — ЧТО ВСТРОЕНО

| Принцип | Где работает | Как |
|---------|--------------|------|
| **Узнавание себя** | Q1, Q3 | Клиент видит свою историю → "это про меня" |
| **Агитация боли** | Q3, Q5 | Касание раны "нет слуха", "уже поздно" |
| **Прошлая неудача = не твоя вина** | Q4 | Снимаем стыд: "пробовал YouTube — не виноват, это не работает 1-на-1" |
| **Визуализация результата** | Q2 | Конкретная сцена (ДР, караоке, дома) — не абстракция |
| **Микро-обязательства** | Q1→Q6 | Каждый ответ = маленькое "да", финал = "записаться" легче |
| **Loss aversion** | После Q6 | "Места на эту неделю: 3 из 8" — реальная редкость |
| **Two-step offer** | Финал | Free diagnostic 60 мин → пакет 8+2 (без давления) |
| **Anchor pricing** | На диагностике | 8 пакет = X₽ за урок, +2 бонусом = реально –20% |
| **Reason why** | В оффере | Бонус "+2 урока" обоснован: «чтобы вы успели почувствовать прогресс» |
| **Социальное доказательство** | Между вопросами | "87% наших студентов выбрали этот вариант" — экраны-переходы |
