## 📊 АНАЛИЗ НАЙДЕННОГО КВИЗА (ЧЕКЛИСТ)

### СТРУКТУРА КВИЗА

```
☐ Количество вопросов
   └─ Оптимально: 3-5 вопросов

☐ Наличие Q0 (фильтр)
   └─ Есть ли вопрос на платежеспособность?

☐ Последовательность вопросов
   └─ Q1 (боль) → Q2 (желание) → Q3 (опыт) → Результат?

☐ Количество вариантов ответа
   └─ На каждый вопрос (обычно 3-5)

☐ Текст вопросов
   └─ Эмоциональный? Специфичный? Понятный?

☐ Формулировка ответов
   └─ Ясные? С примерами? С эмодзи?
```

### ПСИХОЛОГИЯ КВИЗА

```
☐ Персонализация
   └─ Упоминается ли имя клиента в результате?

☐ Эмоциональность
   └─ Вызывает ли эмоции? (страх, надежда, амбиция)

☐ Социальное доказательство
   └─ Есть ли примеры успехов других?
   └─ Есть ли testimoni / отзывы?

☐ Гарантии
   └─ "Результат или деньги назад"?
   └─ "Первый урок бесплатно"?

☐ Urgency (срочность)
   └─ "Осталось 1 место"?
   └─ "Скидка действует 7 дней"?
   └─ "Лимитированное предложение"?

☐ Дизайн и визуалика
   └─ Красивый ли дизайн?
   └─ Есть ли фото педагога?
   └─ Есть ли видео?
```

### КОНВЕРСИОННЫЕ ЭЛЕМЕНТЫ

```
☐ Форма записи после квиза
   └─ Сколько полей? (1-3 = идеально)
   └─ Какие поля? (имя, телефон, email?)

☐ Call-to-action (CTA)
   └─ Какая кнопка? ("Записаться", "Получить результат", "Узнать больше"?)
   └─ Как выглядит? (яркая? заметная?)

☐ Направление потока
   └─ После заполнения формы → куда? (Telegram? Email? Календарь?)
   └─ Есть ли благодарственное письмо?

☐ Интеграция с каналами
   └─ Форма отправляется в CRM?
   └─ Отправляется в Telegram-бот?
   └─ Есть ли email-рассылка?

☐ Гарантия отслеживания
   └─ Даются ли квизы в рекламе? (VK, Instagram, Telegram)
   └─ Видны ли они в соц. сетях?
   └─ Какие каналы работают лучше?
```

## 🎯 ЧТО ИСКАТЬ В КВИЗАХ (САМОЕ ВАЖНОЕ)

### ОБЯЗАТЕЛЬНЫЕ ЭЛЕМЕНТЫ ХОРОШЕГО КВИЗА:

```
✅ ФИЛЬТР ПЛАТЕЖЕСПОСОБНОСТИ (Q0)
   РАБОТАЕТ у всех ТОП школ
   → Уменьшает мусор в 3-5 раз
   → Увеличивает конверсию в оплату в 2+ раза

✅ ЭМОЦИОНАЛЬНЫЙ ВОПРОС (главная боль)
   РАБОТАЕТ у всех ТОП школ
   → Клиент ЧУВСТВУЕТ себя понятым
   → Включается механизм "мне поможешь"

✅ ПЕРСОНАЛИЗИРОВАННЫЙ РЕЗУЛЬТАТ
   РАБОТАЕТ у ТОП школ (90%+)
   → Ребята видят себя в результате
   → Активируется визуализация

✅ ЯСНАЯ CTA + ФОРМА
   РАБОТАЕТ у ТОП школ (100%)
   → Без неясности, что делать дальше
   → Форма МАКСИМУМ 3 поля

✅ СКИДКА / ГАРАНТИЯ / URGENCY
   РАБОТАЕТ у ТОП школ (95%+)
   → "Скидка 30%" или "Результат за 4 неделе"
   → "Осталось 1 место" или "Действует 7 дней"
```

### КРАСНЫЕ ФЛАГИ (ПРИЗНАКИ НЕРАБОТАЮЩЕГО КВИЗА):

```
❌ НЕТ ФИЛЬТРА платежеспособности
   → Контакты будут мусор (20% конверсия)

❌ ВОПРОСЫ слишком РАСПЛЫВЧАТЫЕ
   → "Что вас интересует в пении?" (слишком широко)

❌ РЕЗУЛЬТАТ не ПЕРСОНАЛЬНЫЙ
   → "Вы готовы петь" (одно для всех)

❌ НЕТ ФОРМЫ ЗАПИСИ в квизе
   → Клиент должен сам искать, как записаться

❌ НИКАКОЙ СРОЧНОСТИ
   → Нет скидки, нет гарантии, нет лимита мест

❌ НЕПОНЯТНАЯ CTA
   → "Дальше" вместо "Записаться на урок"
```
